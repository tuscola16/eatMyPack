## App Integration Notes

### Data Fields to Track Per Item
```
id: string
name: string
brand: string
category: enum(gel, bar, chew, drink_mix, real_food, nut_butter, freeze_dried)
serving_size_g: number
serving_size_oz: number
calories: number
cal_per_g: number (calories / serving_size_g)
cal_per_oz: number (calories / serving_size_oz)
carbs_g: number
protein_g: number
fat_g: number
sodium_mg: number
caffeine_mg: number (0 if none)
gut_friendliness: enum(excellent, very_good, good, moderate, poor)
best_for: string[]  // ["early_miles", "aid_station", "night", "high_intensity", etc.]
label_accuracy_note: string  // Important for Spring Energy items
```

### Calorie Density Thresholds (Useful for UX)
- **Ultralight priority** (backpacking): Aim for ≥100 cal/oz carried
- **Good** (backpacking): 80–100 cal/oz
- **Acceptable**: 60–80 cal/oz
- **Heavy** (use fresh at camp): <60 cal/oz

### Weight vs. Volume
- Most energy gels: ~32–60g, very compact (tube or flat packet)
- Chews: 50–60g packet, slightly bulkier
- Bars: 30–85g, most compact per calorie
- Squeeze packs: 32g, flat foil; stackable
- Drink mixes: powder; extremely compact per calorie
- Freeze-dried: 100–170g dry pouch; bulky relative to energy bars
